# LifeOS

LifeOS is a modern, premium, local-first Flutter lifestyle dashboard for tasks, habits, journaling, health, goals, and personal analytics.

## Principles

- Offline-first by default with instant local state updates.
- Feature-first clean architecture.
- Riverpod state management and GoRouter navigation.
- Material 3 light and dark themes.
- Subtle entrance transitions, tap feedback, soft glass cards, and ambient gradient surfaces for a modern premium feel.
- Local persistence layer prepared for Isar, Shared Preferences, and Secure Storage.

## Navigation

The app uses a mobile-first bottom navigation shell with Home, Tasks, Habits, Journal, and Profile destinations plus a floating quick-add action.

## Development

flutter pub get
